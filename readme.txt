# Amar Bangla Keyboard (আমার বাংলা কীবোর্ড)

আমার বাংলা কীবোর্ড একটি বিনামূল্যে ওপেন সোর্স বাংলা টাইপিং সফটওয়্যার।
ইংরেজি অক্ষর দিয়ে বাংলা লিখুন — যেকোনো অ্যাপ্লিকেশনে।

---

## ইনস্টল (সহজ পদ্ধতি)

### পদ্ধতি ১: Setup Installer (সর্বোত্তম)

1. `installer` ফোল্ডারে যান
2. `AmarBanglaKeyboard-Setup-1.0.0.exe` চালান
3. নির্দেশ অনুসরণ করুন

### পদ্ধতি ২: Portable (পোর্টেবল)

1. `dist\Amar Bangla Keyboard.exe` ডাউনলোড করুন
2. যেকোনো ফোল্ডারে রাখুন
3. চালান

### পদ্ধতি ৩: Python থেকে চালানো

```bash
pip install -r requirements.txt
python app.py
```

## ব্যবহার

| কাজ | শর্টকাট |
|------|---------|
| বাংলা/English টগল | **Ctrl+Space** |
| সাহায্য | **F1** |
| বন্ধ করুন | Tray → Right-click → Quit |

**টগল ইন্ডিকেটর:**
- বাংলা মোড = নীল আইকন (বা)
- English মোড = ধূসর আইকন (EN)
